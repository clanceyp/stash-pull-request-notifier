Getting started

This extension displays a queue of STASH pull requests you have been named in.

Setup Bitbucket Server

To display pull requests, the first thing you need to do is login to your Stash/Bitbucket server instance and add your Stash/Bitbucket server URL to the options.

Setub Bitbucket Clound

Put "https://api.bitbucket.org/2.0/" to the Repo. URL field, In the Advanced Settings section
